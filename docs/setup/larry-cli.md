# Larry CLI

Status: in progress.

Larry CLI provides an interactive schema-filling wizard for running generators without wiring MCP into an editor or agent.

## Planned content

- Installation
- Authentication and API key setup
- Starting the CLI
- Selecting a generator
- Filling the schema interactively
- Choosing the output directory
- Saving generated code
- Re-running generation from a schema file

## Outcome

The final version of this guide will show how to use Larry from a terminal-first workflow with minimal setup.
